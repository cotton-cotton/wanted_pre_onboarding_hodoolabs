## ✅ 과제 구현 목록

**_1) Next.js SSR_**

- Next.js로 환경 초기 설정.

**_2) 기능구현_**

- 전체 페이지 UI 구현.
- Custom Hook을 활용한 스크롤 애니메이션 구현.
- 스크롤 위치에 따른 각 컴포넌트 애니메이션 재생.
- slick 라이브러리 설치하여 슬라이드 기능 구현.

## 💻 설치 및 시작하는 법

1. 파일 클론 받기

```
git clone https://github.com/cotton-cotton/wanted_pre_onboarding_hodoolabs.git
```

2. 패키지 설치

```
yarn install
```

3. 실행

```
yarn dev
```
